## Video Planning
### Technical Demonstration Video
1. Show Ollama installed and running locally.
2. Show a terminal command proving the model is available.
3. Show the Unity project open.
4. Explain that Unity sends a local HTTP request to Ollama.
5. Show the request endpoint or `OllamaClient` script.
6. Generate a case.
7. Ask a suspect a custom question.
8. Show the response returned in the game.
9. Briefly mention latency, fallback, and local inference.

### Final Showcase Video
1. Show the game title and main menu.
2. Generate a new case.
3. Read the case briefing.
4. Interrogate suspects.
5. Ask custom questions.
6. Accuse one suspect.
7. Show the result screen.
8. Explain how the LLM improves gameplay.
