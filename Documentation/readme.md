# Project README

## Project Overview
- `CaseFile: Local Suspect` is a Unity 2D detective interrogation prototype with local Ollama integration.

## How To Run The Game
- Open the Unity project.
- Start Ollama locally.
- Open the main scene.
- Press Play.

## Dependencies
- Unity `6000.0.44f1`
- Ollama
- A local model such as `llama3.2`

## Setup Summary
- Full setup steps are tracked in `setup.md`.

## Controls
- Mouse for buttons.
- Keyboard for typing interrogation questions.
